This project shows my work in data cleaning, exploratory data analysis, statistical testing, and machine learning using a retail sales dataset.

Main Work Included
Data cleaning (missing values, duplicates check)

EDA (boxplots, histograms, correlation heatmap)

Discount impact analysis (ANOVA + T‑test)

Delivery performance analysis

Revenue prediction model (Linear Regression)

External dataset merge

Log transformation + outlier handling
